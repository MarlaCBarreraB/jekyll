JEKYLL-DOCS "User Manual"
=================================

NAME:
----

jekyll docs

DESCRIPTION:
-----------

Launch local server with docs for Jekyll

SYNTAX:
------

jekyll docs

COMMANDS:
--------

`-p`, `--port [PORT]`
  Port to listen on

`-u`, `--host [HOST]`
  Host to bind to

ALIAS:
------- 

None
