Turn your text into a monster with Jekyll, the simple, blog-aware, static site generator.

Simple

No more databases, complicated blogging engines, comment moderation, or pesky updates to install. Just your content, the way you want.

Blog-aware

Permalinks, categories, pages, posts, and custom layouts are all first-class citizens here.

Static

Create your content using Markdown (or Textile), Liquid, HTML & CSS, and Jekyll will generate a static site ready for deployment using your favorite web server.

Free hosting with GitHub Pages

Sick of dealing with hosting companies? GitHub Pages is powered by Jekyll, so you can easily deploy your site for free—custom domain name and all.
